How to make an icon uri for Circuit Sandbox

PNG images are created in PowerPoint file "Illustration Fodder.ppt", in the Circuit Siulator icons/ folder

Icons with PowerPoint

  • Draw the icon in PowerPoint, save it as a .png.
  • Open the icon in Preview and resize to 18 x 18 pixels.
  • Add a link to the icon below, in this file.

Icons with Adobe Illustrator

  • Draw the icon in Illustrator at actual scale, save as .svg.
  • During save, uncheck "responsive" (or try both ways)
  • Add a link to the icon below, in this file.

  • right-click on icon image, INSPECT
  • right-click on "img src=somelink", OPEN LINK IN RESOURCES PANEL
  • right-click on image, COPY IMAGE AS DATA URL
  • paste into schematic javascript file as ok_icon.src = 'data:image/png;base64,iVBORw0K ...
